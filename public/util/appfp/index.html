Analizador de Faltas, Apercibimientos y PD

Proposta de proceso de xestión de faltas e apercibimentos.

Realizar unha vez á semana na hora de titoría creando unha carpeta coa data do día en formato "aaaa-mm-dd"

Cómpre ter actualizada a xustificación de faltas.

  1. Obter e gardar o informe de Faltas Detalladas do Xade.
  2. Cargar Informe na APP Faltas e Apercibimentos
  3. Cargar, se é o caso, apercibimentos previos.
  4. Se hai novos apercibimentos ou comunicacións de PD, xerar e gardar o arquivo de apercibimentos actualizado.
  5. Xerar no Xade os informes de apercibimentos e comunicacións PD para o alumnado afectado no punto anterior. Gardar e imprimir
  6. Deixar en secretaría para obter sinaturas e rexistro de saída.
  7. Notificar ao alumnado presencialmente ou vía Abalar (consultar manual de Amalia)
  8. Actualizar o arquivo de apercibimentos coa data de acuse de recibo.

Entrada de Datos

  • DATOS - Módulos.csv
    Datos de los módulos, número de horas anuales, etc. en formato CSV como el modelo que se adjunta. Extraído de https://www.edu.xunta.gal/fp/familias-profesionais
  • DATOS - Apercibimentos.csv (Opcional)
    Datos de Apercibimientos y Comunicaciones de PD en formato CSV como el modelo que se adjunta.
    Si no se aporta se puede omitir y utilizar la propia aplicación para generar y descargar un primer borrador en el que luego sólo habrá que ajustar las fechas de informe y acuse.
  • FALTAS DETALLADAS
    Contenido del Informe predefinido en PDF "AL030-Faltas detalladas" con todas las faltas desde el inicio del curso (9/9/2026). Se puede hacer un copy-paste "a lo bruto" del contenido del PDF en el cuadro de texto o cargar un fichero de texto donde se haya guardado este contenido previamente. La aplicación parseará y filtrará el contenido para procesar sólo las faltas de asistencia no justificadas.

Salida de Datos

  1. Módulos y Umbrales
  2. Faltas Detalladas
  3. Tabla Resumen
  4. Apercibimientos (6%)
  5. Comunicaciones PD (4%)
  6. Informe para imprimir
